Módulo central: utils.py

Único punto de verdad para constantes y funciones compartidas por todos los módulos ads_*.py y app.py.

from utils import (
    ADS_TOKEN,        # Bearer token de NASA ADS
    ADS_API,          # URL base de la API ADS
    STOP_WORDS,       # ~100 palabras vacías (gramaticales + científicas genéricas)
    fetch_arxiv_doc,  # arXiv ID → dict ADS (fl configurable)
    arxiv_to_bibcode, # arXiv ID → bibcode (verbose=False para uso silencioso)
    is_arxiv_id,      # Detecta formato arXiv YYMM.NNNNN
    pubdate_filter,   # Año/rango → filtro ADS pubdate:[...]
)

Herramientas CLI

ads-search — Buscar por autor

ads-search "Apellido, Nombre"
ads-search "Apellido, Nombre" --year 2020-2023
ads-search "Apellido, Nombre" --first-author --export matriz.csv

ads-topics — Buscar por concepto, frase o identificador

--fieldDescripción
all (default)Frase exacta en título o abstract
titleFrase exacta solo en título
abstractFrase exacta solo en abstract
title-wordsTérminos individuales en título (sin stopwords; ideal para títulos completos)
identifierLookup directo: arXiv ID, DOI o bibcode ADS
ads-topics "supernova classification" --year 2024-2026 --rows 10
ads-topics "Optical Observations of SN 2014cx" --field title-words
ads-topics "2301.07688" --field identifier

ads-references / ads-citations

ads-references "2022AJ....164..195F"          # bibcode o arXiv ID
ads-citations  "2022AJ....164..195F" --rows 50 --export matriz.csv

ads-similar — Papers similares

En modo texto, los acrónimos en mayúsculas (GRB, SLSN, ZTF, FBOT…) se preservan antes de normalizar para mejorar la precisión en ADS.

ads-similar --bibcode "2022AJ....164..195F" --year 2020-2026
ads-similar --text "GRB afterglow classification using ZTF photometry"
ads-similar --file mi_abstract.txt --export matriz.csv

ads-chain — Cadena de referencias multinivel

ads-chain "2022AJ....164..195F" --levels 3 --max-per-level 10 --export cadena.csv

ads-download — Descargar PDFs

ads-download "2022AJ....164..195F" --dir ~/papers
ads-download --from-csv mi_matriz.csv --dir ~/papers/matriz

Agente arXiv Diario

python main.py             # ejecución normal
python main.py --dry-run   # sin enviar nada

Flujo: arXiv API → filtro por keywords → digest HTML + MD → WhatsApp (3 reintentos con 2 s de pausa). Lock file .last_run evita reenvíos el mismo día.

Configuración (config.py)

CATEGORIES  = ["astro-ph.HE", "astro-ph.SR", "astro-ph.IM", "cs.LG"]
KEYWORDS    = ["supernova", "GRB", "kilonova", "SLSN", ...]
MAX_RESULTS = 50    # tope de papers por ejecución
HOURS_BACK  = 72    # ventana de búsqueda en horas

Editable desde la interfaz web (panel Configuración) sin tocar el código.
